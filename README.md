# Mind Threads

Minimalist productivity tool designed to visualize thoughts as evolving "threads". Each thread represents a topic or problem and grows vertically through connected updates, creating a timeline-style log.

Designed to free up mental space by writing down persistent thoughts, tracking their progress over time and bringing clarity to mental clutter.

**Live demo:** https://mind-threads.netlify.app/

## Features

- Create threads for ongoing thoughts or problems
- Add updates and actions to track progress
- Archive completed threads
- Local storage persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
